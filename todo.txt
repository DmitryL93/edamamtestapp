Тестовое задание

Написать веб приложение на Vue используя фреймворк Nuxt.
Приложение должно состоять из 1 страницы в виде каталога с поиском и списка элементов.

На странице необходимо подключить обращения к любой публичной API,
можно взять любое API из этого источника: https://rapidapi.com/collection/list-of-free-apis

Обязательно необходимо
использовать flex верстку + SCSS,
предусмотреть адаптивность под любые размеры экранов,
использовать TypeScript.

Функционал:
- поиск по любому совпадению свойств элемента(
  например, это интернет-магазин по продаже цифровой техники,
  товар имеет наименование, цвет, размер и т. д.
  При вводе в поиске любого из свойств товара – фильтровать список, отображая совпадения
)
- изменение отображения списка элементов (
  реализовать 2 вида: плиточный, горизонтальный
)
- добавить пагинацию (
  можно в любой форме
)
- при клике на любой из элементов выводить те самые свойства элемента (детализация).
  Можно использовать модальное окно или переход на отдельную страницу.
- использовать vuex/store, для хранения элементов, действий клиента.

Приветствуется креативность в дизайне, но и плохой дизайн не повлияет на результат если нет возможности.
Использовать готовые UI компоненты разрешается.
По времени не ограничиваем, спешить нет нужды если есть желание проработать задание.
